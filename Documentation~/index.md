# Code Editor Package for Antigravity

## About Antigravity Editor

The Antigravity Editor package provides the Unity Editor with support for Unity-specific features from the [Unity for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=visualstudiotoolsforunity.vstuc) extension in Antigravity. These include IntelliSense auto-complete suggestions, C# editing, and debugging.

## Installation

This package is a built-in package and installed by default.

**Note**: If you're using a version of the Unity Editor before 2019.4, you'll need to install this package through the package manager.

## Requirements

This version of the Antigravity Editor package is compatible with the following versions of the Unity Editor:

* 2019.4 and later

To use this package, you must have the following third-party products installed:

* Antigravity (default path: `/Applications/Antigravity.app`) with [Unity for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=visualstudiotoolsforunity.vstuc) extension 0.9.0 or newer.

## Submitting issues

Submit issues directly from the **Help** menu in Antigravity (**Help** > **Report an Issue**).
